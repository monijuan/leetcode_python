### 343. 整数拆分

给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。

示例 1:

输入: 2
输出: 1
解释: 2 = 1 + 1, 1 × 1 = 1。
示例 2:

输入: 10
输出: 36
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36。
说明: 你可以假设 n 不小于 2 且不大于 58。

class Solution:
    def __init__(self):
        pass

    def cuttingRope(self, n: int) -> int:
        if n <= 3: return n - 1
        a, b = n // 3, n % 3
        if b == 0: return 3**a
        elif b == 1: return 4*3**(a-1)
        else:return 2*3**a


def test(data_test):
    s = Solution()
    return s.cuttingRope(*data_test)


def test_obj(data_test):
    result = [None]

        result.append(res)
