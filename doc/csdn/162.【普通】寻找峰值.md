### 162. 寻找峰值

峰值元素是指其值严格大于左右相邻值的元素。

给你一个整数数组 nums，找到峰值元素并返回其索引。数组可能包含多个峰值，在这种情况下，返回 任何一个峰值 所在位置即可。

你可以假设 nums[-1] = nums[n] = -∞ 。

你必须实现时间复杂度为 O(log n) 的算法来解决此问题。

 

示例 1：

输入：nums = [1,2,3,1]
输出：2
解释：3 是峰值元素，你的函数应该返回其索引 2。
示例 2：

输入：nums = [1,2,1,3,5,6,4]
输出：1 或 5
解释：你的函数可以返回索引 1，其峰值元素为 2；
     或者返回索引 5， 其峰值元素为 6。
 

提示：

1 <= nums.length <= 1000
-231 <= nums[i] <= 231 - 1
对于所有有效的 i 都有 nums[i] != nums[i + 1]

from leetcode_python.utils import *


class Solution:
    def __init__(self):
        pass

    def findPeakElement_暴力遍历(self, nums: List[int]) -> int:
        length = len(nums)
        if length==1:return 0
        elif length==2:
            if nums[0]>nums[1]:return 0
            else:return 1
        for id,num in enumerate(nums):
            if id==0 and num>nums[1]:return id
            elif id==length-1 and num>nums[-2]:return id
            elif nums[id-1]<num and num>nums[id+1]: return id


def test(data_test):
    s = Solution()
    return s.findPeakElement(*data_test)


def test_obj(data_test):
    result = [None]
    obj = Solution(*data_test[1][0])
    for fun, data in zip(data_test[0][1::], data_test[1][1::]):
        if data:
            res = obj.__getattribute__(fun)(*data)
        else:
            res = obj.__getattribute__(fun)()
        result.append(res)
    return result


if __name__ == '__main__':
